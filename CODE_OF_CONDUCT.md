# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
Read the full text at the link above.

**In plain English:**

1. Be welcoming. Different skill levels, backgrounds, and opinions
   make the project better.
2. Be direct without being unkind. Critique the code, not the
   person.
3. Assume good faith. Most miscommunications are not malice.
4. No harassment, slurs, or personal attacks — anywhere in this
   repository's issues, PRs, discussions, or commits.
5. If something feels off, contact the maintainer.

**Reporting:**

Open a [private security advisory](https://github.com/StevenWang-CY/cortex/security/advisories/new)
(GitHub keeps advisories private until the maintainer resolves them).
For non-sensitive concerns, open a regular issue.

The maintainer reserves the right to remove comments, close issues,
and block accounts that violate this code.
