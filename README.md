# Ralph Project

A project powered by [Ralph for Claude Code](https://github.com/frankbria/ralph-claude-code) — autonomous AI development framework.

## Getting Started

1. Edit `.ralph/PROMPT.md` — describe your project
2. Add specs to `.ralph/specs/` — define what to build
3. Run `ralph --monitor` — watch Ralph build it
